# OpenDAW Quick Start Guide

Get up and running with OpenDAW in 5 minutes.

---

## Installation

### Windows
1. Download `OpenDAW-Setup-1.0.0.exe` from the releases page
2. Run the installer (includes Visual C++ Redistributable)
3. Launch OpenDAW from the Start Menu or Desktop shortcut

### macOS
1. Download `OpenDAW-1.0.0.dmg`
2. Drag OpenDAW to Applications
3. Launch from Applications folder

---

## First Launch

On first launch, OpenDAW will:
1. Show a welcome screen
2. Offer to load the demo project
3. Run an audio engine test

**Recommended:** Accept all three to verify your setup.

---

## Creating Your First Project

### 1. Create a New Project
- File → New Project (or Ctrl+N)
- Choose a location for your `.opendaw` project folder
- Set your sample rate (48kHz recommended)

### 2. Add Tracks
- Click the "+" button in the track header
- Choose track type: Audio or MIDI
- For audio: drag files from your computer or Suno library

### 3. Import from Suno Library
- Click the Suno icon in the sidebar
- Browse your library (20 tracks shown by default)
- Drag a track to an empty clip slot

### 4. Record MIDI
- Connect a MIDI controller
- Click the red record button
- Play notes on your controller
- Click stop to end recording

### 5. Export Your Track
- File → Export → Audio
- Choose format: WAV 24-bit (recommended) or MP3 320kbps
- Select export range: Entire project or loop selection
- Click Export

---

## AI Features

### Stem Separation
1. Right-click any audio clip
2. Select "Extract Stems"
3. Wait for processing (progress shown)
4. 4 new tracks appear: Vocals, Drums, Bass, Other

### AI Pattern Generation
1. Right-click empty clip slot
2. Select "Generate Pattern"
3. Choose style: Electronic, House, Techno, Ambient, Jazz
4. Set tempo, key, and length
5. Click Generate
